Notes App - Kotlin + SQLite

Aplicación de notas desarrollada en **Kotlin** con **Android Studio**, utilizando **SQLite** como base de datos local.  
Permite crear, editar y eliminar notas con una interfaz moderna basada en **Material Design**.

Características

-  Crear, editar y eliminar notas.  
- Almacenamiento persistente local (SQLite).  
- Interfaz moderna con **Material Design**.  
- Listado de notas en un RecyclerView.  
- Compatible con Android 8.0 (API 26) en adelante.

Tecnologías utilizadas

- **Kotlin**
- **Android Studio**
- **SQLiteOpenHelper**
- **Material Components**
- **RecyclerView**
